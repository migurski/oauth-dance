Dance The OAuth Dance

Doing the OAuth dance can be an incredible pain
when you just need to get data out of an API.
Let us dance the dance on your behalf.

How To

Set up a new application with one of the providers below,
be sure to set your callback URL to this site, and enter your
OAuth client ID and secret in this form.

{{callback_url}}

Don’t trust us? Good.
Install the app yourself using Python code from Github.

Github

New application:
https://github.com/settings/applications/new

OAuth documentation:
https://developer.github.com/v3/oauth/
